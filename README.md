🚦 Smart Control of Traffic Light using Artificial Intelligence

A practical simulation of an intelligent traffic light system that adapts signal timing based on traffic conditions.

## 📌 Project Overview

Traditional traffic signals use fixed timers, which leads to unnecessary waiting and congestion.

This project introduces a **smart traffic control system** that dynamically adjusts traffic signals using simple AI logic and visual simulation.

## 🎯 Objectives

- Reduce traffic congestion  
- Minimize waiting time at signals  
- Simulate real-time traffic control  
- Provide a visual and interactive system  

## 🧠 How It Works

1. Traffic video is used as input  
2. The system overlays:
   - Traffic lights (Red, Yellow, Green)
   - Timer countdown
   - Count Vehicles
3. Signal timing is controlled dynamically  
4. Future scope includes vehicle detection using AI models  

## 🛠️ Technologies Used
- Python  
- OpenCV  
- NumPy
